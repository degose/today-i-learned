* meta 태그
    * HTML 문서가 어떤 내용을 담는지, 문서의 키워드는 무엇인지, 누가 만들었는지 문서 자체의 특성을 담는다.
    * head태그 안에 위치
    * name, content 등의 속성 => 검색엔진에게 문서의 내용을 요약해주는 역할
    * `<meta http-equiv="refresh" content="5;url=http://www.homejjang.com/“>` => 5초뒤 url 속성값으로 지정한 페이지로 이동
